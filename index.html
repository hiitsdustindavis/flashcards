Javascript

Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Modulo

Modulo

(in number theory) with respect to or using a modulus of a specified number. Two numbers are congruent modulo a given number if they give the same remainder when divided by that number.

Operators

Operators

Constructs which behave generally like functions, but which differ syntactically or semantically from usual functions.

NaN

NaN

In computing, NaN, standing for not a number, is a numeric data type value representing an undefined or unrepresentable value, especially in floating-point calculations.

Variable Naming Conventions

Variable Naming Conventions

A set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation.

Function

A block of code designed to perform a particular task.

Method

Method

The actions that can be performed on objects.

Argument to a Method

Argument to a Method

An argument is the expression of the parameter contained within the method.

String

String

Any finite sequence of characters (i.e., letters, numerals, symbols and punctuation marks).

Chaining Methods

Chaining Methods

Chaining allows us to run multiple jQuery methods (on the same element) within a single statement.

Escape

Escape

This function makes a string portable, so it can be transmitted across any network to any computer that supports ASCII characters.

Alert

Alert

The alert() method displays an alert box with a specified message and an OK button.

Commenting Code

Commenting Code

Characters readable to the programmer but ignored by the browser via specific opening and closing tags.

Booleans

Booleans

In computer science, the Boolean data type is a data type, having two values (usually denoted true and false), intended to represent the truth values of logic and Boolean algebra. It is named after George Boole, who first defined an algebraic system of logic in the mid 19th century.

Undefined

Undefined

The undefined property indicates that a variable has not been assigned a value.

Parameter

Parameter

Function parameters are the names listed in the function definition. Function arguments are the real values passed to (and received by) the function.

Return

Return

The return statement ends function execution and specifies a value to be returned to the function caller.
